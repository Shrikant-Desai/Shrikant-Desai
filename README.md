## 🚀 Featured Projects

### 🧠 AI Resume Analyzer & Job-Fit Scorer
Full-stack app that scores resume-to-job-description fit using LLM extraction and vector similarity.

**Key Features**
- Uploads a resume + JD; Gemini extracts structured skills and generates a **Job Readiness Score (0–100)** with verdict tiers
- Cosine similarity on skill embeddings (pgvector) classifies each JD requirement as matched / partial / missing
- Redis caching to cut redundant LLM API calls on repeat analyses
- Zero-downtime deployment on ECS Fargate behind API Gateway with auto-scaling

**Tech Stack:** `React.js` `Next.js` `Python` `FastAPI` `PostgreSQL` `pgvector` `Redis` `Gemini API` `Docker` `AWS ECS` `Grafana`

**Impact:** Automated skill-gap analysis end-to-end, with full observability via Grafana dashboards for latency, errors, and container health.
